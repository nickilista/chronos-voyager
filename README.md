# Chronos Voyager

A Vibe Jam 2026 entry.

Pilot a time-traveling starship through historical eras. Dodge obstacles. Solve
math puzzles at each portal to unlock the next era. Built with Three.js,
deployed on Vercel.

## Dev

```
npm install
npm run dev      # http://localhost:5173
npm run build    # production build -> dist/
```

## Stack

- TypeScript + Vite
- Three.js + postprocessing
- Vercel (chronosvoyager.com)
